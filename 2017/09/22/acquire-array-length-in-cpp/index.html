C++获取数组长度

一、静态数组。

比如 int int_ary[5] ,静态数组在编译的时候就知道了数组长度，这对编译器来说挺好实现的。

用得最多的大概就是sizeof了吧：

#define countof_macro(x) (sizeof((x)) / sizeof(x)[0])

对于C++，可以用模板。
